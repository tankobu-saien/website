やさいの育て方

ここでは野菜の育て方や育ち方の一例をお伝えします。

  • ナス科
  • ウリ科
  • マメ科
  • バラ科

〇ミニトマト

思っているよりたくさん収穫するのが難しい野菜です。

<原産地>　南米アンデス山脈
<発芽適温>15℃
<栽培適温>15℃～27℃

苗は倒して植えます。根っこが増えるので生育初期は水やり控えめ。
奇数果房の花が実に代わり始めたら肥料を少し。配合肥料を与えてもよい。
収穫は朝。糖代謝が起こる前の収穫がおいしい。

〇ナス

手をかけるほど答えてくれる野菜です

<原産地>　インド
<発芽適温>18℃
<栽培適温>20℃～30℃

いかに根を広げられるか、生育初期から株の体力を温存しておくこと。肥料食いではあるが、配合肥料などを使いつつ、メリハリをつける。
剪定を行いましょう。焦げナス防止のためだけでなく、良質なものを収穫できます。
平行になっている枝、株の中心になっている枝を収穫後に空きます。

〇ピーマン

苗選びが大事

<原産地>　中南米
<発芽適温>20℃
<栽培適温>20℃～30℃

苗半作って聞いたことある？？
ホームセンターや園芸店で苗の成長点を見るとしわしわということがあります。
7月ぐらいで実が終了してしまいます。
株のしっかりした苗を選びましょう。育て方はナスと変わらないです。
万願寺トウガラシやししとうにも応用が効きます。

〇いちご

初心者でも育ててみよう

<原産地>　北米東部、南米チリ
<発芽適温>種からは向かない
<栽培適温>15℃～25℃

家でできたらかわいいし嬉しいですよね。１０月ごろに苗が出回る一季なりイチゴがおすすめです。品種は自分が食べたい品種を選びましょう。
（とちおとめがおすすめ（笑））植えたら過乾燥に注意しながら冬越しし梅の花と同時に花を咲かせます。追肥は化成肥料少々と、バットグアノがおすすめ！
